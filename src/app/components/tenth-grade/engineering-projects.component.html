An AP Seminar presentation that I did. The presentation was about how the physical and non-physical factors of prison culminate into a prisoner's well being, and their ability to reintegrate into society.

Skills I learned:
- Creating good slide designs to engage an audience
- Keeping composure even when in a stressful situation, such as presenting

Individual Written Argument

This was a part of AP Seminar, that involved me writing an argumentative paper that argued that the physical and non-physical factors of a prison culminated into a prisoner's well being and their ability to reintegrate into society.

Skills I learned:
- Showing a range of perspectives around a topic to display an argument
- Using JSTOR as a place for sources of books and academic journals
This is one of the most effective papers I've written for a school Language Arts project. It incorporates many different arguments which are backed by academic sources of evidence.

Digital Assistant QA Chatbot

This was an engineering makerspace project, where I made a physical chatbot using a raspberry pi that when given a question prompt, it would respond with an answer to the prompt.

Skills learned:
- Configuring audio on Linux
- Using the pip module chatterbot, to create a chatbot and train it
- Using the pip modules speech_recognition and pyttsx3 for text-to-speech and speech-to-text
This project showcases me effectively using code to create a tangible product in the real world.

Planetary Gearbox

This was a team makerspace project, where we created a planetary gearbox, a gear system that has a central/large gear, surrounded by smaller gears. When the large gear turned, all of the small gears would turn as well.

Skills learned:
- Learning how to use an arduino for electronics projects
- Learning how to use a digital multimeter to measure electrical output
I used engineering to help solve real-world problems with gearing systems, and designing of them.

Flower Stands

This was another team engineering makerspace project where we created these test tube holders, in which these test tubes would hold these flowers. This was for a wedding of one the teachers.

Skills learned:
- Using a drill press to drill holes into a piece of wood
- Using a air gun to clear particles off of an object
