My 9th Grade Projects

Balloon Car

Description:
This was an engineering project where I teamed up with a few people to design and build a car that would be propelled forward using a balloon.
The purpose of this project was to meet the requirements specified in the document.

Skills learned:
- I learned how the coating of a certain surface, such as a balloon, affects the friction of a moving object, such as a car.
- I learned also how to use citation generators, such as BiBTeX
The document showcasing this project clearly outlines the requirements of the project, clearly outlines improvements that may have been needed to be made to the design, and helps a person who might be new to the project understand what we were trying to do.

CorelDRAW Keychain

This was an engineering project where we were supposed to design a keychain using CorelDRAW, a software for creating 2d graphics. We were also allowed to use onshape DXF's imported into CorelDRAW for our keychain designs.

Skills learned:
- I learned how to use CorelDRAW to create 2d graphics to be laser cut
This project overall showcases me properly using CorelDRAW to create a keychain design, which was not merely a random shape to be cut.

Mean Temperature Predictor

A showcase project for my initial portfolio website for engineering, that predicts the mean temeprature of given day given its high and low temperature.

Skills learned:
- I learned how to use the linear regression model from sklearn to predict an outcome using a dataset.
- I learned how to choose the best data columns of a dataset to have a model yield an accurate result.
Deriving predictions from a dataset is very important in fields such as finance, healthcare, and etc. This project showcases me effectively utilizing some data from a dataset to generate new predictions about the mean temperature of a certain day.

Iris Species Classifier

Another showcase project for my engineering portfolio site that predicts a species of a certain Iris species flower given mutliple parameters such as petal length, and etc.

Skills learned:
- I learned how to tokenize a label column of a dataset (e.g. encoding one output to a value of 0, another to a value of 1, and etc.)

Classification is an important tool in technologies such as computer vision, and identifying certain types of disease. This project showcases me classifying flowers from random data about them.
